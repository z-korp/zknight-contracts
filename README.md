## Launching SC Locally

To launch SC locally, follow these steps:

1. **Launch Katana**:

```
katana --disable-fee
```

2. **Build with Sozo**:

```
sozo build
```

3. **Migrate with Sozo**:

```
sozo migrate
```

4. **Start Torii**:

```
torii --world <@address>
```

Please remember to replace `<@address>` with the appropriate address when using the command.
